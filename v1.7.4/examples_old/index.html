Examples

One-dimensional case

The Su-Schriffer-Heeger (SSH) model

Here's a simple example of the SSH Hamiltonian:

julia> using TopologicalNumbers
julia> function H(k) # set SSH Hamiltonian function of wavenumber k
    g = 0.9
    
    [
        0 g+exp(-im*k)
        g+exp(im*k) 0
    ]
end

The band structure is computed as follows:

julia> showBand(H; value=false, disp=true)

In this case, 1 signifies the dimension of the wavenumber space.

Next, we can calculate the winding numbers using calcBerryPhase:

julia> calcBerryPhase(H)

The output is:

(TopologicalNumber = [1, 1], Total = 0)

The first argument TopologicalNumber in the named tuple is an vector that stores the winding number for each band. The vector is arranged in order of bands, starting from the one with the lowest energy. The second argument Total stores the total of the winding numbers for each band (mod 2). Total is a quantity that should always return zero.

One-dimensional phase diagram is given by:

julia> function H0(k, p)
            [
                0 p[1]+p[2]*exp(-im * k)
                p[1]+p[2]*exp(im * k) 0
            ]
        end
julia> H(k, p) = H0(k, (p, 1.0))

julia> param = range(-2.0, 2.0, length=1001)
julia> calcPhaseDiagram(H, param, "BerryPhase"; plot=true)

Also, two-dimensional phase diagram is given by:

julia> param = range(-2.0, 2.0, length=101)
julia> calcPhaseDiagram(H0, param, param, "BerryPhase"; plot=true)

Chern numbers

Two-dimensional square lattice with flux model

A two-dimensional example is presented here:

julia> function H(k) # landau
    k1, k2 = k
    t = 1

    Hsize = 6
    Hmat = zeros(ComplexF64, Hsize, Hsize)

    for i in 1:Hsize
        Hmat[i, i] = -2*cos(k2-2pi*i/Hsize)
    end

    for i in 1:Hsize-1
        Hmat[i, i+1] = -t
        Hmat[i+1, i] = -t
    end

    Hmat[1, Hsize] = -t*exp(-im*k1)
    Hmat[Hsize, 1] = -t*exp(im*k1)
    
    Hmat
end

To calculate the dispersion, run:

julia> showBand(H; value=false, disp=true)

Then we can compute the Chern numbers using calcChern:

julia> calcChern(H)

The output is:

(TopologicalNumber = [1, 1, -2, -2, 1, 1], Total = 0)

The first argument TopologicalNumber in the named tuple is an vector that stores the first Chern number for each band. The vector is arranged in order of bands, starting from the one with the lowest energy. The second argument Total stores the total of the first Chern numbers for each band. Total is a quantity that should always return zero.

One-dimensional phase diagram is given by:

julia> function H(k, p)
    k1, k2 = k
    t = p

    Hsize = 6
    Hmat = zeros(ComplexF64, Hsize, Hsize)

    for i in 1:Hsize
        Hmat[i, i] = -2 * cos(k2 - 2pi * i / Hsize)
    end

    for i in 1:Hsize-1
        Hmat[i, i+1] = -t
        Hmat[i+1, i] = -t
    end

    Hmat[1, Hsize] = -t * exp(-im * k1)
    Hmat[Hsize, 1] = -t * exp(im * k1)

    Hmat
end

julia> param = range(-2.0, 2.0, length=500)
julia> calcPhaseDiagram(H, param, "Chern"; plot=true)

$\mathbb{Z}_2$ numbers

The Bernevig-Hughes-Zhang (BHZ) model

As an example of a two-dimensional topological insulator, the BHZ model is presented here:

julia> function H(k) # BHZ
    k1, k2 = k
    t = 1

    R0 = -2(cos(k1) + cos(k2)) + 1
    R3 = 2sin(k2)
    R4 = 2sin(k1)
    R5 = -2t*(cos(k1) + cos(k2)) + 1

    s0 = [1 0; 0 1]
    sx = [0 1; 1 0]
    sy = [0 -im; im 0]
    sz = [1 0; 0 -1]

    a0 = kron(s0, s0)
    a1 = kron(sx, sx)
    a2 = kron(sx, sy)
    a3 = kron(sx, sz)
    a4 = kron(sy, s0)
    a5 = kron(sz, s0)

    R0*a0+R3*a3+R4*a4+R5*a5
end

To calculate the dispersion, execute:

julia> showBand(H; value=false, disp=true)

Next, we can compute the $\mathbb{Z}_2$ numbers using calcZ2:

julia> calcZ2(H)

The output is:

(TopologicalNumber = [1, 1], Total = 0)

The first argument TopologicalNumber in the named tuple is an vector that stores the $\mathbb{Z}_2$ number for each each pair of two energy bands. The vector is arranged in order of bands, starting from the one with the lowest energy. The second argument Total stores the total of the $\mathbb{Z}_2$ numbers for each pair of two energy bands. Total is a quantity that should always return zero.

Total is a value that should consistently return zero.
